# Maze_Solver

Generates mazes using a [Recursive backtracker](https://en.wikipedia.org/wiki/Maze_generation_algorithm)
and solves mazes using the [A* Algorithm](https://en.wikipedia.org/wiki/A*_search_algorithm).
